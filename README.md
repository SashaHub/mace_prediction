# A repository with sample code for predicting negative outcomes of cardiac events based on tabular and unstructured text data.
* [MACE prediction](https://github.com/SashaHub/mace_prediction/blob/main/Models_VEGFR2.ipynb): Prediction of negative cardiac outcomes based on tests and genetic markers.
* [BioMistral](https://github.com/SashaHub/mace_prediction/blob/main/BioMistral_adaptation.ipynb): Domain adaptation of BioMistral using PEFT and quantization for MACE classification task on EHR data.
* [MedGemma](https://github.com/SashaHub/mace_prediction/blob/main/MedGemma_adaptation.ipynb): Domain adaptation of MedGemma using PEFT and quantization for MACE classification task on EHR data.
